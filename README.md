# Paradise Nursery 🌿

**Paradise Nursery** is an online plant shop built with React and Redux, allowing users to browse houseplants, add them to a shopping cart, and manage their purchases.

## Project Overview

This is a single-page e-commerce application for an online plant shop. Users can explore a curated collection of houseplants grouped by category, add items to a cart, adjust quantities, and proceed to checkout.

## Features

- Browse 18+ unique houseplants across 3 categories
- Add plants to a shopping cart with a single click
- Cart icon dynamically updates with total item count
- Responsive navigation bar with links to Home, Plants, and Cart
